# JSMeow (BETA)

### NodeJS for external Game Hacking (Windows only)

### Requirement:

```text
- NodeJS v16.14.0 or latest (x64 only)
- CMAKE-JS
- CMAKE
```

### Installing:

```bash
NPM: npm install @droidxrx/jsmeow
YARN: yarn add @droidxrx/jsmeow
```
